# alert-fullscreen

### normal behaviour
Chrome, Firefox and Safari:
window goes fullscreen, alert gets displayed after 2 seconds

### Edge issue
window goes fullscreen, alert is hidden

See: https://github.com/microsoft/MicrosoftEdge-Extensions/issues/558#issuecomment-4333783093

### Orion issue
window does not go fullscreen, alert is visible

Tested on Orion Version 1.0.6 (145), WebKit 624.1.7, Tahoe (26)

See: https://orionfeedback.org/d/13694-chromewindowsupdatewindowid-state-fullscreen-fails
